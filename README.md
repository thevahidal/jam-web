
# Jam 
The Jam web application 


## Run Locally
Clone the project
```bash
  git clone https://github.com/thevahidal/jam-web
```

Go to the project directory
```bash
  cd jam-web
```

Install dependencies
```bash
  yarn
```

Start the server
```bash
  yarn dev
```
